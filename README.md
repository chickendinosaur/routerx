# Development  

## Installation  

* git clone https://github.com/chickendinosaur/routerx.git
* cd routerx
* npm install

## Build  

* npm run build

## Benchmarking  

* npm run benchmark

## Test  

* npm run test

## Publish  

* npm run deploy
